** JavaScript Display Imkoniyatlari

-------------------------------------------
 innerHTML - HTML elementlari ichiga yozadi.
-------------------------------------------

-----------------------------------------------------------------------------
 document.write() - HTML ning chiqishiga yozadi.
 
 HTML document o'qib sahifa yuklanib bo'lgandan keyin document.write() 
 ishlatilsa to'liq HTML o'chirilib uning o'rniga yangi qiymat yozib qo'yiladi.
 
 document.write() metod faqat test qilish uchun ishlatilishi kerak.
-----------------------------------------------------------------------------

-----------------------------------------------------
 windows.alert() - Alert box yozishda.
 
 Ma'lumotni alert box qilib ko'rstishda ishlatiladi.
 window.alert() or alert()
 Biz window metodini o'tkaizb yuborishimiz ham mumkin.
-----------------------------------------------------

------------------------------------------------------------------
 console.log() - Browserning consoliga biror biz ma'lumotni yozadi.
------------------------------------------------------------------
